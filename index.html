SASS, Paso a Paso

  1. Descargar e instalar Node.js.
  2. Entrar en la consola o terminal y posicionarse en la carpeta del proyecto donde queremos inicializar SASS.
  3. Ejecutar en orden los siguientes comandos:
    • npm init
    • Responder todas las preguntas apretando enter. Cuando nos pregunta escribir Yes
    • Escribir en consola: npm install -D node-sass nodemon
  4. La carpeta css ya la tienen creada con su archivo adentro, ahora tienen que crear una carpeta scss con un archivo scss.css
  5. Abrir package.json para poder editarlo.
  6. Pegar la línea de buil.css y watch.css debajo de "test" (agregar una "," al final de la línea).
    • "build-css": "node-sass --include-path scss scss/style.scss css/estilo.css", "watch-css": "nodemon -e scss -x \"npm run build-css\""
  7. Ejecutar el último comando y empezar a trabajar. Escribir en consola npm run watch-css para que el style.scss actualice el style.css. A partir de ahora cada vez que guarde el archivo correrá en terminal el comando npm run watch-css.
